# 📋 점수제 산정 기준 (Points-Based System)
F-2 비자나 자격 취득을 위한 점수 산정 체계를 바탕으로 합니다.
상세 점수 항목 및 합격 기준은 다음과 같습니다.

## 1. 주요 특징 (Scoring Items)

각 항목은 **항목, 최대점수, 세부 기준** 요소로 구성되어 있습니다.
합격 기준은 총점 80점 이상(170점 만점) 입니다.

### 🔢 점수 항목 상세

| 항목 | 최대점수 | 세부 기준 |
| :------ | :------ | :------: |
| **공통항목** | **130** |  | 
| **나이** | **25점** | 25-29세: 25점, 18-24/30-34세: 23점, 35-39세: 20점 등 |
| **학력** | **25점** | 박사: 25점, 석사: 20점, 학사: 15점 등 |
| **한국어 능력** | **20점** | TOPIK/KIIP 5급 이상: 20점, 4급: 15점 등 |
| **연간 소득** | **60점** | 1억 이상: 60점, 9천만: 58점, 6천만: 50점 등 |
| **가점항목** | **40점** | 국내 학위(15), 연속체류3년(10), 자격증(10), 추천서(15), 봉사/세금(5씩) |
| **감점항목** | **-70점** | 출입국법 위반(-20/회), 국내법 위반(-30/회) |

---

## 2. 체류 기간 (Duration of Stay)

> [!IMPORTANT]
> 1. **130점+:** 5년 
> 2. **120-129점:** 3년 
> 3. **110-119점:** 2년 
> 4. **80-109점:** 1년 

---

## 🛠️ 활용 방법
1. 본인의 상황에 맞는 항목별 점수를 합산합니다.
2. 감점 항목이 있다면 최종 점수에서 차감합니다.
